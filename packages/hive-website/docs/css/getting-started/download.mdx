## Hive CLI

Hive CLI will prompt a few questions and depending on the answers, it will download and install the relevant packages.

1. Run the following command:

```bash
npx hive-ui-init
```

2. Choose which **Hive CSS framework** you wish to use in your project:

<img src={useBaseUrl('img/docs/01-hive-cli.png')} />

3. Choose which **Hive UI framework platform** you wish to use in your project:

<img src={useBaseUrl('img/docs/02-hive-cli.png')} />

4. If you chose **Web Components**, select now which **JavaScript framework** you are using:

<img src={useBaseUrl('img/docs/03-hive-cli.png')} />
