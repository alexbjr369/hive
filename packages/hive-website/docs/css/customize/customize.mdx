> **There are multiple ways to customize Hive CSS framework. It will depend on your project, build tools, browser support, and more.**

## Overview

It is recommend the use of a package manager so you can extend Hive's source files.

## CSS Precompilers

### File structure

Whenever possible, avoid modifying Hive's core files. Create your own stylesheet that imports Hive so you can modify and extend it.
Assuming you're using a package manager, you'll have a file structure that looks like this:

### Importing

In your `hive-custom.{css,less,sass,scss,styl}`, import Hive's source files.

> **Check Hive's [`playground project`](https://github.com/alexbjr369/hive/tree/main/packages/hive-playground/javascript/css). In this example project, [Rollup](https://rollupjs.org/guide/en/) and [TypeScript](https://www.typescriptlang.org/) were used.**

```css title="hive-custom.css"
:root {
  /* variables overrides */
}
```

  </TabItem>
  <TabItem value="less">

> **Check Hive's [`playground project`](https://github.com/alexbjr369/hive/tree/main/packages/hive-playground/javascript/less). In this example project, [Rollup](https://rollupjs.org/guide/en/) and [TypeScript](https://www.typescriptlang.org/) were used.**

```less title="hive-custom.less"
// 1. hive required stylesheets
@import '../node_modules/@alexbjr369/hive-less/src/variables.less';

// 2. variables overrides
@import './variables-overrides.less';

// 3. remainder of hive required stylesheets
@import '../node_modules/@alexbjr369/hive-less/src/functions.less';
@import '../node_modules/@alexbjr369/hive-less/src/mixins.less';

// 4. optional components
@import '../node_modules/@alexbjr369/hive-less/src/layout/container.less';

// 5. autoral imports
@import './autoral.less';
```

  </TabItem>
