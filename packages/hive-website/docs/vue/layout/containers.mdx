# Containers

> **Containers are the most basic building block in Hive, they align and pad the content within a given viewport.**

## Features

**Links:** [`Storybook example`](https://alexbleggi.netlify.app/hive/stories/vue/?path=/story/layout-containers--overview) [`Figma design`](https://www.figma.com/file/1qFtanhiPCXP6zQRw3fgWn/?node-id=35%3A3)

Containers are **required** when using Hive's default grid system.

## Markup

### Default Container

> **A full width container, spanning the entire width of the viewport.**

```tsx
<hv-container>100% wide at all breakpoints</hv-container>
```

**Links:** [`Storybook example`](https://alexbleggi.netlify.app/hive/stories/vue/?path=/story/layout-containers--default) [`Figma design`](https://www.figma.com/file/1qFtanhiPCXP6zQRw3fgWn/?node-id=113%3A741)

### Responsive Containers

> **Responsive containers allow you to specify a modifier that is 100% wide until the specified breakpoint is reached, after which a `max-width` is applied for each of the higher breakpoints.**

```tsx
<hv-container size="sm">100% wide until small breakpoint</hv-container>
<hv-container size="md">100% wide until medium breakpoint</hv-container>
<hv-container size="lg">100% wide until large breakpoint</hv-container>
<hv-container size="xl">100% wide until extra large breakpoint</hv-container>
<hv-container size="xxl">100% wide until extra extra large breakpoint</hv-container>
```

**Links:** [`Storybook example`](https://alexbleggi.netlify.app/hive/stories/vue/?path=/story/layout-containers--responsive) [`Figma design`](https://www.figma.com/file/1qFtanhiPCXP6zQRw3fgWn/?node-id=113%3A852)

### Fixed Container

> **A responsive and fixed-width container, its `max-width` changes at each breakpoint.**

```tsx
<hv-container size="fixed">width changes at each breakpoint</hv-container>
```

**Links:** [`Storybook example`](https://alexbleggi.netlify.app/hive/stories/vue/?path=/story/layout-containers--fixed) [`Figma design`](https://www.figma.com/file/1qFtanhiPCXP6zQRw3fgWn/?node-id=113%3A921)

Its possible to modify the container `max-width` in a given breakpoint:

```css {4} title="variables-overrides.css"
:root {
  --hv-container-width-sm: 540px;
  --hv-container-width-md: 720px;
  --hv-container-width-lg: 900px;
  --hv-container-width-xl: 1140px;
  --hv-container-width-xxl: 1320px;
}
```

Modify the container `padding`:

```css {2} title="variables-overrides.css"
:root {
  --hv-container-padding: 1.5rem;
}
```

Modify the container `padding` for a given breakpoint:

```css {4} title="variables-overrides.css"
:root {
  --hv-container-padding-sm: 1rem;
  --hv-container-padding-md: 1rem;
  --hv-container-padding-lg: 2rem;
  --hv-container-padding-xl: 1rem;
  --hv-container-padding-xxl: 1rem;
}
```

Its possible to modify the container `max-width` in a given breakpoint:

```less {4} title="_variables-overrides.less"
@container-widths: {
  sm: var(~"--@{namespace}-container-width-sm, 540px");
  md: var(~"--@{namespace}-container-width-md, 720px");
  lg: var(~"--@{namespace}-container-width-lg, 900px");
  xl: var(~"--@{namespace}-container-width-xl, 1140px");
  xxl: var(~"--@{namespace}-container-width-xxl, 1320px");
};
```

Add a new container `max-width` in a custom breakpoint:

```less {8,17} title="_variables-overrides.less"
@breakpoints: {
  xs: 0;
  sm: 576px;
  md: 768px;
  lg: 992px;
  xl: 1200px;
  xxl: 1400px;
  custom: 1600px;
};

@container-widths: {
  sm: var(~"--@{namespace}-container-width-sm, 540px");
  md: var(~"--@{namespace}-container-width-md, 720px");
  lg: var(~"--@{namespace}-container-width-lg, 960px");
  xl: var(~"--@{namespace}-container-width-xl, 1140px");
  xxl: var(~"--@{namespace}-container-width-xxl, 1320px");
  custom: var(~"--@{namespace}-container-width-custom, 1560px");
};
```

Modify the container `padding`:

```less {1} title="_variables-overrides.less"
@container-padding: var(~"--@{namespace}-container-padding, 1.5rem");
```

Modify the container `padding` for a given breakpoint:

```less {4} title="_variables-overrides.less"
@container-paddings: {
  sm: var(~"--@{namespace}-container-padding-sm, @{container-padding}");
  md: var(~"--@{namespace}-container-padding-md, @{container-padding}");
  lg: var(~"--@{namespace}-container-padding-lg, 2rem");
  xl: var(~"--@{namespace}-container-padding-xl, @{container-padding}");
  xxl: var(~"--@{namespace}-container-padding-xxl, @{container-padding}");
};
```
